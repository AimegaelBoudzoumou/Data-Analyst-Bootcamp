# How to use DAX in Power BI

### Count of sales
```dax
Count of Sales = COUNT('Apocolypse Sales'[Order ID])
```

### Sum of Products Sold
```dax
Sum of Products Sold = SUM('Apocolypse Sales'[Units Sold])
```
