# Comparison Operators

|   Operator |   Name |
|:-:    |:-:    |
|   ==   |   Equal |
|   !=  |   Not equal |
|   >   |   Greater than |
|   <   |   Less than |
|   >=   |   Greater than or equal to |
|   <=   |   Less than or equal to |


## __==__ : Equals
## __!=__ : Not Equal
## __>__ : Greater than
## __<__ : Less than
