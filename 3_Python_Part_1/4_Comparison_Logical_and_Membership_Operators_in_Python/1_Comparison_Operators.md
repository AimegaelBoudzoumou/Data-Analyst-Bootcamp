# Comparison Operators

|   Operator |   Name |
|:-:    |:-:    |
|   ==   |   Equal |
|   !=  |   Not equal |
|   >   |   Greater than |
|   <   |   Less than |
|   >=   |   Greater than or equal to |
|   <=   |   Less than or equal to |

```python
10 = 10 # SyntaxError: cannot assign to literal

10 == 10 # True

10 == 50 # False

10 != 50 # True

'Vanilla' == 'Chocolate' # False

'Vanilla' == 'Vanilla' # True

'Vanilla' == 'vanilla' # False

x = 'Vanilla'

y = 'Chocolate'

x == y # False

x != y # True

10 < 50 # True

10 < 10 # False

10 <= 10 # True

50 => 10 # True
```
